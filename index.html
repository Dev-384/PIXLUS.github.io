Game Setup

Installing a few dependencies to be able to host a local server. Using a server allows for more complex Javascript, using imports and exports between javascript files

To be able to play, you will need to do the following:

Install Node.js

Go to Download Node.js

PIXLUS uses NodeJS to host the local server, which allows for imports and exports between javascript files. Without this, you would be unable to play PIXLUS

Open PIXLUS

In a terminal, make sure to be inside the PIXLUS directory.
For example:

>

Run the following command:

> npm run main

This will install other needed dependencies, to allow for the hosting of the PIXLUS server. After this is done, the server will start. The terminal will show you whwere to play. The installed dependencies are placed inside the main PIXLUS folder, inside the directiory "node_modules" These are only used to host the server, they do not have any threat to your device.

Playing PIXLUS

After running the terminal commands, a message will show appear inside the terminal with the following message:

Server is running on port (3000) Veiw the server at: (http://localhost:3000/web/) Closing this terminal will result in the termination of the server. The following messages will be server messages/errors. __________________________________________________

This message tells you the location of the web server. In this case, the server is located at http://localhost:3000/web/ (Line 2 in terminal message). This is the URL to go to for playing PIXLUS

Without this terminal open, you will not have access to play PIXLUS, the terminal is keeping the server up and running.
